About MyTodo App

MyTodo is a simple Flask-based web application that helps you manage your daily tasks efficiently.

Features

  • Add new todos with title and description
  • View all saved todos in a table
  • Update existing todos
  • Delete completed or unwanted todos

How to Use

  1. Add Todo
    Fill the Todo Title and Description fields and click Submit.
  2. View Todos
    All your todos will appear in the table below the form.
  3. Update Todo
    Click the Update button to edit a todo.
  4. Delete Todo
    Click the Delete button to remove a todo.

Technology Used

  • Python (Flask Framework)
  • SQLite Database
  • HTML, CSS, Bootstrap 5

Created as a learning project to understand Flask CRUD operations.
